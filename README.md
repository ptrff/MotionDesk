### Features  
- Combine images
- Apply and customize effects
- Gesture control
- User-friendly interface
- Save live wallpapers
- Edit saved wallpapers

### Tech stack

 - Java
 - LibGDX
 - GDX-VFX
 - GLSL
 - Gson
 - RxJava2
 - Room ORM
 - Picasso
